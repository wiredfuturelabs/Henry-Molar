HELP CENTER

  • Element Info

    • Click element in name to view info about a element.Scroll the data to view more information.Click a specific column to highlight the column.

  • Calculate

    • Click the button in the bottom of status bar to open the compound calculator . Type element/compound name to search wikipedia for the details about it. Type the compound formula and choose an option to view info about the compound.

  • Dark theme

    • Press the moon button(found in status bar) to view the app in dark button.

  • Report error

    • Press the warning button (found in status bar) to report any kind of error in the app.

  • Feedback

    • Send feedback of the app by clicking the feedback button (found in status bar).

  • Help

    • Press the icon with question mark(found in status bar) to open this help center.
